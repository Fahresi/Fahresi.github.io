Tugas

Keterangan Tugas Apa Saja yg Telah Di buat pada Tugas ini

Tugasnya adalah Membuat Game di Unity

Langkah-Langkah :

1. Buat New Project pada Unity Hub, Gunakan Universal 2D / 2D, Dan Jadikan Local Project

2. Download Assets yg telah di berikan oleh Pak Guru

3. Setelah itu Masukkan Assets di unity dengan cara Drag File Ke dalam Unity

4. Pada Unity Klik Import

6. Klik Mobile Fluppy Bird Lalu liat Pada Inspector Bar, Bagian Sprite Editor Ganti dari Single Jadi Pultiple, bagian Pixel Per Unit Ganti menjadi 32, Lalu Pencet Sprite Editor

7. Pada Sprite Editor Tekan Slice, Type Automatic dan Slice Lalu Apply

8. Nanti Akan Muncul Seperti Di Atas

9. Tambahkan Background Dan Ground

10. Di Background Pada Inspector Bar Di Sprite Renderer Ubah Order In Layer Menjadi -1, Agar Backgroundnya berada di Layer Paling Bawah

11. Untuk Menambahkan Player Pencet Tombol Shift Lalu Pilih Player Berurutan Sesuai Dengan Animasi, Dan Masukkan Ke dlm Scene

12. Bikin New Folder Dengan Nama Animation Dan Save Dengan Nama Player

13. Rename Tiap Assets yg telah di taro

14. Klik Kanan Lalu Create Empty rename jadi Player Dan Masukkan Player Kedalamnya

15. Pada Player Add Component,Lalu Tambahkan RigidBody 2D, Dan Capsule Colider 2D

16. Pencet Edit Colider Lalu Sesuaikan Sesuai Bentuk Player

17. Klik Kanan Pada Hierarcy Lalu Create Empty, Ganti Namanya Menjadi Pipa

18. Tambahkan Pipa Atas Dan Bawah, Rename menjadi Pipa 1 dan 2, Lalu Masukkan Ke dalam Pipa yg sdh di Buat

19. Tekan Shift Dan Pilih Pipa1 dan Pipa2 Untuk Groupings

20. Tambahkan Component Capsule Colider 2D pada Pipa
